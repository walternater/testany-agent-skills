# Video Script Writer

启动短视频脚本创作工作流。为产品推广短视频生成完整的制作脚本。

## 使用方式

提供 BRD 选题编号或主题描述：

$ARGUMENTS

## 支持的平台

- **YouTube** - 2–3 分钟完整版
- **抖音 / TikTok** - 60–90 秒精华版

## 6 阶段工作流

1. **Researcher** - 选题调研（痛点、竞品、演示功能点）
2. **Hook Writer** - 前 3 秒钩子创作（3–5 个方案）
3. **Scriptwriter** - 完整分镜脚本（时间轴 + 画面 + 口播）
4. **Voiceover Formatter** - ElevenLabs 配音稿格式化
5. **Storyboard Designer** - 录屏清单 + Remotion 动画规格
6. **Reviewer** - 审核终稿 + 制作清单

## 阶段推进

每个阶段完成后会暂停等待你的批准。用自然语言推进即可：

- "继续" / "下一步" — 进入下一阶段
- "开始 Hook 创作" — 直接指定进入 Stage 2
- "写脚本" — 进入 Stage 3
- "进度如何" — 查看当前工作流状态

如果没有提供选题，请告诉我你想为哪个主题制作视频。
