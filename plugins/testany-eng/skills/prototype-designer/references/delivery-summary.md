### 对下游的输入

**对 API Contract**：
- [原型暴露的数据需求——哪些页面需要什么数据、什么结构]

- [实时性需求（如需 WebSocket）]

**对 HLD**：
- [状态管理复杂度（跨页面共享状态？）]
- [缓存需求（哪些数据需要缓存？）]
- [性能敏感点（大数据量页面、高频交互）]
